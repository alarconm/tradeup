## Production Notes

### ElevenLabs Settings
- Voice: "Rachel" (clear, professional female) or "Josh" (friendly male)
- Stability: 50%
- Clarity: 75%
- Style: 0%
- Speed: Slightly slower than default (0.9x)

### Screen Recording Settings (OBS)
- Resolution: 1920x1080
- Frame Rate: 60fps
- Browser Window: Maximized, no bookmarks bar
- Clean up any test data that looks messy

### Timing Guide for Recording
- Pause 2 seconds at each major screen before moving
- Let animations complete before clicking next
- Move mouse smoothly, not jerky
- Hover over elements being discussed

### Post-Production (CapCut)
- Add subtle zoom on key UI elements
- Lower background music during voiceover
- Add captions (CapCut auto-generates them)
- Fade in/out transitions between scenes
